# Invernadero

> *¡Ese nombre está sujeto a cambios!*

El proyecto **Invernadero** busca desarrollar
un invernadero inteligente que pueda
cuidarse a si mismo, la idea radica el concepto
de cuidado autónomo a su vez que busca fomentar
el conocimiento cultural mediante diseños que
hagan alución a elementos culturales (como especies
de aves únicas de la zona)
